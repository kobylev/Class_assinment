# דרישת תרגיל ליצירת ולניתוח מערך נתונים בינארי

## רקע
יש ליצור מערך דו-ממדי בגודל 15 עמודות על 5,000 שורות. בכל תא ימוקם ערך בינארי אקראי (0 או 1). כל עמודה מייצגת תכונה (למשל תכונות של חייזרים), אך שמות העמודות אינם חשובים - ניתן לקרוא להן A, B, C וכדומה.

## דרישות מרכזיות:
- ליצור מערך בגודל 15x5000 הממולא בערכי 0 ו-1 אקראיים.
- לחשב עבור כל זוג עמודות את מדדי Support ו-Confidence בעזרת הנוסחאות:
  - **Support**: אחוז השורות שבהן שני האירועים מתרחשים בו זמנית (כלומר, שני הערכים הם 1).
  - **Confidence**: היחס בין Support של שני האירועים לבין Support של האירוע הראשון.
- למצוא את כל הקומבינציות של זוגות עמודות (X → Y) שעוברות את הקריטריונים הבאים:
  - תמיכה (Support) של לפחות 30%
  - ביטחון (Confidence) של לפחות 70%
- יש לחשב את המדדים באופן ידני, ללא שימוש בספריות מוכנות שמבצעות את החישובים אוטומטית.
- מותר להשתמש בספריות כגון numpy ו-pandas (עם התייחסות זהירה לאופן ההמרה בין סוגי הנתונים).
- מומלץ להגביל את חיפושי הקשרים לקומבינציות קטנות (לדוגמה זוגות ולא שלשות ומעלה) כדי למנוע חיפוש נרחב מדי.
- ניתן לשדרג את אחוז האחדים בעמודות במטרה לשפר את איכות הקשרים שיימצאו.

## תהליך העבודה המומלץ:
1. בניית מערך הנתונים האקראי.
2. יישום פונקציות לחישוב Support ו-Confidence בין עמודות.
3. סריקת כל זוגות העמודות במערך למציאת כל הקשרים העומדים בדרישות.
4. הצגת התוצאות עם פירוט: השמות והאינדקסים של העמודות, Support ו-Confidence.
